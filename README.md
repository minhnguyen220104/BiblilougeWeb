# BiblilougeWeb
E-Commerce
This Website is built for following purpose:-

For buying product online
Maintaining sale history
Adding and managing products
User Friendly
For Implemention of Generic Servlets in Java
This is a Mini-project developed using Java, Jdbc, Bootstrap.
Admin Have Following Access for this E-Commerce website:-

Add New Products
Add New Category
View Available Products
View Current Users
View Category
Remove Products
Users Have Following Access for this E-Commerce website:-

Create New Account or Register
Login
View Available Products
Select Product to Buy
Select Product Quantity
Add To cart
Go to Checkout Page
Technologies used:-
Front-End Development:
Html
Css
Javascript
Bootstrap 4
Back-End Development
Java
JDBC
Servlet
Hibernate
Maven Support
Database used
Sql/MySql
